## 七、为什么不能放链接/二维码？替代策略是什么？

### 官方规则理解

官方说的"不得出现二维码/站外链接等营销引流"，核心判断逻辑是：
- **直接引流**：放网址、二维码、"点击链接"、"私信获取" → 违规
- **间接引流**：暗示"加我微信"、"关注我公众号"、"去XX平台找我" → 也可能违规
- **软引流**：只展示内容本身，让用户主动搜索 → 相对安全，但灰色地带

**建议：完全不放任何引流信息，纯展示内容 + 求互动**。因为比赛的核心目标是人气分（点赞+评论×2+收藏+转发），不是真的引流。

### 替代策略：如何让人找到你的作品？

既然不能在抖音内容里放链接，那初赛专区的帖子链接怎么办？可以这样做：

| 渠道 | 合规操作 | 目的 |
|------|---------|------|
| **抖音内容本身** | 只展示项目，不放任何链接，只引导"点赞助力" | 拿人气分 |
| **初赛专区帖子** | 正常放体验链接（这是官方允许的） | 让评审体验 |
| **小红书/B站** | 可以发项目介绍，软植入"正在参加TRAE大赛"，评论区可以适度放链接（小红书相对宽松） | 外部引流 |
| **朋友圈/微信群** | 直接发初赛专区链接，让朋友去帖子体验 + 来抖音点赞 | 熟人裂变 |
| **技术社区** | V2EX/即刻/推特发项目实现，软植入"抖音参赛作品"，不直接放抖音链接 | 技术圈助力 |

### 核心逻辑

**抖音这条内容 = 比赛人气分工具**，不是引流工具。
- 家长/路人看到 → 觉得有价值 → 点赞/评论/收藏 → 人气分增加
- 真的想体验的人 → 会主动搜索"TRAE 创造力大赛"或"安心童伴" → 找到初赛专区
- 朋友帮你 → 直接在抖音里点赞评论，不需要体验链接

所以**不要纠结"怎么在抖音里放链接"，要纠结"怎么让内容本身打动人，让他们愿意点赞"**。
