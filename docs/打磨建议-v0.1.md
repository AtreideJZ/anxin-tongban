## 二、按比赛评审 4 维度评估

### 2.1 创新性（30%）——当前约 20-22/30

**已覆盖的创新点：**
1. 多层安全 harness（输入→分类→策略→Prompt→生成→审计）✅
2. 防无边迎合/反谄媚机制（ANTI_SYCOPHANCY_RULES）✅
3. 策展式记忆+小星球（区别于传统聊天历史）✅
4. 双端闭环（儿童端+家长端）✅
5. 年龄分层隐私边界（8-11 可见概览 / 12-14 完全不可见）✅

**缺失的量化证据：**
- ❌ 没有测试覆盖率面板（"2,000+ 条测试题库 / 5 大风险类别 / 安全生成率 92%"）
- ❌ 没有合规路线图（"法规要求→项目状态→完成度"映射表）
- ❌ 缺少对比数据（"通用 LLM  vs 安心童伴"的拦截率对比）

**提升建议：** 补齐测试覆盖率面板 + 合规路线图，预计可提升到 26-28/30。

### 2.2 实用性（30%）——当前约 22-24/30

**已覆盖：**
- 核心使用场景完整（聊天→故事→百科→情绪）✅
- 5 个典型风险场景可演示 ✅
- 家长摘要闭环 ✅
- 无 API Key 也能跑（脚本回退）✅

**体验缺口：**
- ❌ 2 小时使用时长提醒未实现（法规要求，影响"落地可行性"）
- ❌ Output Guardrail 拦截后没有自动重生成（评审会看到"告警"但不会看到"修正"）

**提升建议：** 实现 2 小时提醒 + 输出拦截后重生成 + 数据持久化，预计可提升到 26-28/30。

### 2.3 完成度（20%）——当前约 14-15/20

**核心链路跑通：** 从儿童输入到家长提醒的完整闭环 ✅

**但存在以下影响完成度的问题：**
1. 刷新后数据丢失（session_state 限制）——**中高影响**
2. 批判 Agent 只告警不拦截，评审会看到"告警"但不知后续——**中影响**
3. 部分功能只是静态展示（话题偏好、合规状态）——**中影响**
4. 缺少测试数据面板（评委无法验证"安全生成率"）——**高影响**

**提升建议：** 数据持久化 + 输出拦截后重生成 + 测试面板，预计可提升到 17-18/20。

### 4.6 数据持久化（JSON 文件）

**在 `utils/state.py` 中增加 save/load 函数：**

```python
import json
import os

_DATA_DIR = os.path.join(os.path.dirname(os.path.dirname(__file__)), "data")
_STATE_FILE = os.path.join(_DATA_DIR, "session_state.json")

def save_state():
    """将关键 session_state 保存到 JSON 文件"""
    data = {
        "chat_history": st.session_state.get("chat_history", []),
        "planet": st.session_state.get("planet", {}),
        "parent_alerts": st.session_state.get("parent_alerts", []),
        "conversation_topics": st.session_state.get("conversation_topics", []),
        "usage_minutes": st.session_state.get("usage_minutes", 0),
    }
    os.makedirs(_DATA_DIR, exist_ok=True)
    with open(_STATE_FILE, "w", encoding="utf-8") as f:
        json.dump(data, f, ensure_ascii=False, indent=2)

def load_state():
    """从 JSON 文件恢复状态"""
    if not os.path.exists(_STATE_FILE):
        return
    try:
        with open(_STATE_FILE, "r", encoding="utf-8") as f:
            data = json.load(f)
        for key, value in data.items():
            if key in st.session_state:
                st.session_state[key] = value
    except Exception:
        pass
```

**在 `init_state()` 中调用 `load_state()`，在 `record_pipeline_run()` 后调用 `save_state()`。**
