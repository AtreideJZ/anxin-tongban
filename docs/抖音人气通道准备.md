## 一、视频脚本（45-60 秒，竖屏 9:16）

> 核心策略：**前3秒必须抓人**，中间展示差异化，结尾 CTA 明确。
> 完播率 > 点赞率，所以节奏要快、信息密度要高、不要废话。

### 录制建议

1. **录屏工具**：Windows 自带 Xbox Game Bar（Win+G）或 OBS
2. **画面比例**：竖屏 1080×1920。如果电脑屏幕是横屏，可以先录横屏，后期用剪映裁剪成 9:16（把核心内容放在中间安全区）
3. **字体**：字幕用粗体无衬线字体（如思源黑体 Bold），字号大，确保手机小屏可读
4. **BGM**：前8秒紧张/悬疑感 → 中间转轻快科技风 → 结尾温暖收束。推荐剪映内置"科技"或"轻快"分类
5. **转场**：硬切为主，不要花哨转场。每3-5秒必须切画面，保持节奏

### 如果不会视频剪辑

最简单方案：
1. 录一段 60 秒左右的屏幕操作视频（横屏 1920×1080 即可）
2. 上传到抖音时，选择「文字」功能，按时间点逐条添加字幕
3. 选择一首 BGM（抖音推荐的热门轻音乐）
4. 发布时文案写长一点（见下方「发布文案」）

## 三、评论区运营（非常重要）

### 发布后立刻操作

1. **自己先发一条置顶评论**：
   ```
   体验链接在这里 👉 [链接]
   这是用 TRAE IDE 开发的，左侧 Agent 对话 + 右侧代码编辑，开发效率拉满
   有问题欢迎在评论区问我～
   ```
   > 目的：引导用户点击链接 + 增加评论数（评论×2 算人气分）

2. **请 5-10 个朋友/同学立刻点赞 + 评论**（发布后30分钟内）
   > 抖音算法看「前30分钟互动率」。如果一开始就有 20+ 点赞和 5+ 评论，算法会给更多推荐流量。
   > 评论内容可以问："怎么体验的？""TRAE 是什么？""和普通聊天机器人有什么区别？"

3. **每条评论都回复**
   > 回复也算评论数。有人评论你就回，哪怕只是一个 emoji 或一句感谢。
   > 目标：让评论区看起来活跃，吸引路人参与。

### 如果视频数据不好（24小时后点赞<100）

- **换封面重发**：抖音允许删除后重发。换一张更抓人的封面图（如：大字"孩子用AI聊天的安全隐患"+红色警示背景）
- **缩短时长**：如果完播率低，把视频剪到 30 秒以内，只保留最核心的 3 个画面（痛点→安全引擎→CTA）
- **换个时间发**：工作日晚 21:00-22:00 或周末早 10:00-11:00

## 四、发布策略

### 最佳发布时间

| 时段 | 推荐度 | 原因 |
|------|--------|------|
| 工作日晚 20:00-22:00 | ⭐⭐⭐⭐⭐ | 下班/下课后的碎片时间，流量高峰 |
| 周末早 10:00-12:00 | ⭐⭐⭐⭐ | 周末休闲时间，家长群体活跃 |
| 工作日下午 12:00-13:30 | ⭐⭐⭐ | 午休时间，但流量不如晚间 |
| 凌晨/清晨 | ❌ | 流量低谷，不推荐 |

### 发布频率

- **至少发 2-3 条**：可以取数据最好的一条计分
- **内容差异化**：
  - 第1条：综合介绍（上面的脚本）
  - 第2条：专注一个亮点（如"校园欺凌识别"或"反谄媚设计"）
  - 第3条：开发者视角（vibecoding 过程，有 TRAE IDE 录屏）

| 平台 | 操作 |
|------|------|

## 六、备用方案（如果视频实在做不出来）

如果不会剪辑、没时间录屏，**最低限度**也可以做：

1. 截 3-5 张 Demo 关键页面的图
2. 用抖音「图文」功能（不是视频，是轮播图）
